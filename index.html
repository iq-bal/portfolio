Projects
It was an app made for my Advanced Programming lab projects.

Name for the application was Virtual Cr.

Some of the key features this app offers:

-See upcoming schedules for class test, assignment and lab test

-See class routine for today

-Communicate with classmates

-Search classmates by blood group

-Hall of fame, where everyone celebrates for a friend's success

-Send love on friend's birthday

-Calculates attendance at the end of semester

The UI very vibrant in color, There is a philosophy behind it. Every morning you wake and see a boring black and white class routine kills mood for joining into courses. Colorfull UI sets mood for even boring task in some way.

It was fun working on it.

Hope to work on such projects again real quick
